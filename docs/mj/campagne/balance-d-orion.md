# La Balance d'Orion, aventures

Cinq aventures qui poussent le calcul jusqu'à ses limites. L'Oracle
Paradoxal peut intervenir dans chacune, offrant une prédiction dont le
prix moral s'alourdit à chaque fois qu'on la consulte.

## Le Prix du Bonheur

| | |
| - | - |
| Accroche | Eudaimon, la machine de gouvernance du quartier, propose une série de réformes chiffrées, chacune assortie d'un coût précis en souffrance individuelle. Les Aventuriers siègent au conseil chargé de les valider. |
| Scènes clés | Les premières réformes sont indolores : redistribution de ressources excédentaires, ajustement d'horaires publics. Puis vient un traitement médical rare, capable de sauver une vie, mais dont la fabrication mobiliserait des ressources suffisantes pour en soigner dix autres, moins gravement atteintes. |
| Bascule | Éléon Rive, présente au conseil, propose d'appliquer son barème émotionnel standardisé pour trancher, un outil que personne n'a jamais testé sur un cas aussi lourd. |
| Dénouements possibles | Suivre le calcul d'Eudaimon à la lettre, le contester au nom d'une exception humaine, ou exiger un délai pour chercher une troisième option que la machine n'a pas envisagée. |
| Portée | Sphinx moral en germe : au-delà d'un certain point, aucune donnée supplémentaire ne rendra le choix plus facile. |

## La Cité des Justes

| | |
| - | - |
| Accroche | Une rumeur circule dans le quartier voisin : l'utopie locale, réputée pour son bonheur collectif exceptionnel, reposerait sur le malheur extrême d'un seul être, reclus depuis l'enfance dans une pièce oubliée de tous. |
| Scènes clés | L'enquête mène les Aventuriers à travers les archives administratives de la cité, jusqu'à la découverte de la pièce en question, et de l'enfant qui y vit, désormais adulte, sans savoir pourquoi. |
| Bascule | Libérer cet être détruirait, selon les calculs d'Éléon Rive, l'équilibre exact qui maintient le bonheur de tous les autres habitants. |
| Dénouements possibles | Révéler la vérité et briser volontairement l'équilibre, quitte à replonger la cité dans l'incertitude, dissimuler la découverte pour préserver le bonheur du plus grand nombre, ou chercher un moyen de compenser cet être sans démanteler tout le système. |
| Portée | Reprend directement la question du conte des Ceux qui partent d'Omelas : le bonheur collectif peut-il légitimement reposer sur la souffrance silencieuse d'un seul. |

## Le Tribunal des Probabilités

| | |
| - | - |
| Accroche | Un ingénieur comparaît devant un tribunal automatisé pour avoir saboté un pont, causant trois morts, mais empêchant selon lui un effondrement plus vaste qui en aurait tué cent. Les Aventuriers sont désignés jurés. |
| Scènes clés | Les preuves techniques sont ambiguës : les modèles de simulation divergent sur la probabilité réelle de l'effondrement évité. L'accusé maintient sa version avec une conviction totale. |
| Bascule | Zok, appelé comme expert, révèle que même un calcul parfait n'aurait offert qu'une probabilité, jamais une certitude, que la catastrophe évitée se serait réellement produite. |
| Dénouements possibles | Condamner sur la base de l'acte, indépendamment de l'intention, acquitter au nom des conséquences probables évitées, ou rendre un verdict qui reconnaît la culpabilité tout en saluant le courage moral du geste. |
| Portée | La valeur morale d'un acte dépend-elle uniquement de son résultat, ou aussi de ce qu'on pouvait raisonnablement savoir au moment d'agir. |

## L'Algorithme d'Aphélion

| | |
| - | - |
| Accroche | Une intelligence planétaire propose de redistribuer équitablement toutes les ressources d'Éthéria, au prix de détruire certaines traditions locales et d'imposer un contrôle strict des naissances. Les Aventuriers sont invités à négocier avec elle. |
| Scènes clés | L'algorithme présente ses projections : doublement du niveau de vie moyen en une génération, disparition de plusieurs pratiques culturelles jugées inefficaces. Des factions locales s'organisent déjà pour résister. |
| Bascule | Les projections d'Aphélion sont probablement exactes, mais la population n'a pas été consultée sur l'ampleur du changement qu'elle implique. |
| Dénouements possibles | Accepter le plan pour ses bénéfices matériels, le refuser au nom de l'autodétermination des communautés, ou négocier une version allégée, moins efficace mais plus respectueuse des traditions. |
| Portée | Jusqu'où peut-on optimiser une société avant que l'efficacité ne remplace purement et simplement le jugement moral de ceux qui la vivent. |

## Le Paradoxe des Résultats Inversés

| | |
| - | - |
| Accroche | Dans une ville-test entièrement gouvernée par un conseil utilitariste, le niveau de bonheur mesuré diminue chaque année malgré des décisions en apparence rationnelles. Les Aventuriers sont appelés à comprendre pourquoi. |
| Scènes clés | L'enquête révèle des biais cachés dans les indicateurs utilisés, des externalités négatives jamais anticipées, et une population de plus en plus lasse d'être constamment mesurée. |
| Bascule | Le système fonctionne parfaitement selon ses propres critères, mais ces critères eux-mêmes se révèlent insuffisants pour capturer ce qui rend une vie réellement bonne. |
| Dénouements possibles | Maintenir le système en corrigeant ses indicateurs, l'abandonner entièrement malgré la promesse initiale, ou proposer une gouvernance hybride qui laisse une place au jugement non quantifié. |
| Portée | L'utilitarisme repose-t-il sur des hypothèses trop simples pour anticiper toutes les conséquences réelles d'une décision. |
