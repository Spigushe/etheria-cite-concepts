# Quêtes transversales

Ces quatre aventures traversent plusieurs quartiers à la fois,
remettant en question des logiques qu'on croyait cloisonnées.

## Le Pacte du Paradoxe

| |  |
| - | - |
| Accroche | Une entité venue d'au-delà des concepts établis, sans nom fixe, cherche à fonder un huitième quartier d'Éthéria, bâti sur la contradiction elle-même plutôt que sur un principe unique. |
| Scènes clés | Elle demande aux Aventuriers de réunir des fragments de dilemmes irrésolus, vécus dans chacun des sept quartiers existants. Chaque quartier oppose une résistance différente : le Labyrinthe craint de perdre ses contradictions au profit d'un lieu qui les absorbe, la Concorde redoute un précédent politique incontrôlable. |
| Bascule | Une fois les fragments réunis, le nouveau quartier commence à se former, mais son existence même semble déjà déstabiliser l'équilibre fragile entre les sept quartiers historiques. |
| Dénouements possibles | Achever la fondation du huitième quartier et assumer ses conséquences sur l'équilibre global, refuser de livrer le dernier fragment et laisser le projet inachevé, ou négocier avec l'entité une forme atténuée, un simple lieu de passage plutôt qu'un quartier à part entière. |
| Portée | La contradiction n'est pas nécessairement une impasse : elle peut aussi devenir un moteur de pensée, à condition d'accepter qu'Éthéria n'a pas besoin d'un équilibre parfait pour continuer d'exister. |

## Le Manuscrit Delié

| |  |
| - | - |
| Accroche | Un ancien grimoire conceptuel a été dispersé en fragments à travers les sept quartiers, chacun réécrivant à sa manière un même épisode fondateur de l'histoire d'Éthéria, en totale contradiction avec les autres versions. |
| Scènes clés | Retrouver chaque fragment suppose de composer avec son porteur : un archiviste du Labyrinthe qui refuse de choisir entre les versions, un juriste de la Concorde qui exige une reconnaissance officielle du fragment qu'il détient, un habitant du Gouffre qui n'y voit qu'une preuve supplémentaire de l'absurdité de toute mémoire. |
| Bascule | Une fois tous les fragments réunis, il devient évident qu'aucune version ne peut être vraie sans contredire au moins deux autres, et que le grimoire lui-même semble avoir été conçu pour ne jamais offrir de synthèse. |
| Dénouements possibles | Choisir une version et la déclarer officielle au risque de trahir les autres témoignages, tenter une synthèse qui préserve des fragments de chaque version, ou publier l'ensemble du grimoire tel quel, contradictions incluses. |
| Portée | Toute mémoire est interprétée ; la réconciliation ne passe peut-être pas par l'unification des faits, mais par l'acceptation assumée de la pluralité des récits. |

## Les Gardiens s'effacent

| |  |
| - | - |
| Accroche | Sans avertissement, les Gardiens des sept quartiers deviennent muets, ou se retirent purement et simplement de leurs lieux, laissant les portails sans surveillance et les habitants inquiets. |
| Scènes clés | L'enquête des Aventuriers révèle des causes différentes selon les quartiers : un Gardien semble avoir été oublié par ceux qu'il questionnait, un autre paraît avoir été rejeté à la suite d'un choix collectif récent, un troisième donne l'impression d'avoir simplement dépassé le concept qu'il incarnait. |
| Bascule | Ramener un Gardien à sa fonction d'origine supposerait de restaurer artificiellement une croyance ou une pratique que la population a, de son propre chef, laissé s'éteindre. |
| Dénouements possibles | Tenter de raviver les anciens Gardiens malgré la perte de sens qu'elle implique, laisser émerger de nouveaux Gardiens correspondant à ce que les habitants croient réellement aujourd'hui, ou accepter que certains quartiers restent, pour un temps, sans surveillance. |
| Portée | Les idées ne sont pas éternelles : leur pertinence dépend de leur capacité à continuer de féconder une pensée vivante, pas d'un titre ou d'une fonction figée. |

## L'Accord Inconscient

| |  |
| - | - |
| Accroche | Une force invisible semble orienter simultanément les choix collectifs de plusieurs quartiers, une volonté diffuse qui influence les débats, les votes, et jusqu'aux dilemmes proposés aux Aventuriers eux-mêmes. |
| Scènes clés | En remontant les fils de cette influence, ils découvrent un réseau de symboles partagés, reproduits sans concertation apparente d'un quartier à l'autre, comme si une même intuition collective s'exprimait indépendamment en plusieurs lieux à la fois. |
| Bascule | La source de cette cohésion, une fois localisée, se révèle bienveillante dans ses intentions, mais agit sans que personne ne l'ait explicitement choisie ni même perçue. |
| Dénouements possibles | Dévoiler publiquement cette influence et laisser chacun décider d'y consentir ou de la rejeter, la laisser opérer discrètement puisqu'elle semble faire du bien, ou chercher à la neutraliser par principe, au nom de la lucidité collective. |
| Portée | L'autonomie éthique suppose la lucidité : même une harmonie bienveillante peut étouffer la pensée si elle s'impose sans qu'on la voie venir. |
